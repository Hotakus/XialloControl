# Privacy Policy

**XialloControl** (the "Application") is a locally-running game controller
mapping utility developed and maintained by the project owner.

## Data Collection

**XialloControl does not collect, store, or transmit any personal
information.**

All processing occurs entirely on your local device:

- **Controller input**: Button and axis data from connected game controllers
  is read via standard operating system APIs. This data is processed in
  real-time for mapping purposes and is never recorded or stored.

- **Keyboard and mouse simulation**: The Application simulates keyboard and
  mouse events based on user-defined mappings. All simulated input is
  generated and consumed locally.

- **Local configuration**: User preferences (mapping profiles, dead zone
  settings, etc.) are saved as local files on your device and are never
  uploaded to any server.

## Network Access

The only network request made by the Application is to check for software
updates via the [Tauri updater](https://tauri.app). This request fetches a
version manifest from GitHub Releases and does **not** include any user
data, device identifiers, or telemetry.

## Third-Party Sharing

Since no data is collected, no information is shared with or sold to any
third party.

## Changes to This Policy

If this policy changes in the future, the updated version will be published
at this URL. Continued use of the Application after changes constitutes
acceptance of the revised policy.
